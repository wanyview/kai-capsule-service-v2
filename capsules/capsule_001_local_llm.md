## 内容

### 1. 什么是本地大模型？

本地大模型是指在本地服务器或电脑上运行的大型语言模型，不依赖云端API。

优势：
- 数据安全：数据不出网
- 成本低廉：一次性投入，长期使用
- 响应快速：本地推理延迟低
- 定制化：可针对场景微调

### 2. 为什么选择Ollama？

Ollama是当下最流行的本地大模型运行平台：

- 一键安装
- 支持海量模型
- 简单易用
- 跨平台支持

### 3. 安装步骤

#### Mac系统
```bash
# 安装Ollama
curl -fsSL https://ollama.com/install.sh | sh

# 验证安装
ollama --version
```

#### Linux系统
```bash
# 安装Ollama
curl -fsSL https://ollama.com/install.sh | sh
```

### 4. 模型选择

推荐配置：

| 模型 | 参数 | 适用场景 |
|------|------|----------|
| Qwen2.5 | 7B | 日常对话、知识问答 |
| Llama3 | 8B | 通用任务 |
| DeepSeek | 7B | 代码、推理 |

### 5. 快速开始

```bash
# 拉取模型
ollama pull qwen2.5:7b

# 启动模型
ollama run qwen2.5:7b

# 测试对话
>>> 你好，请介绍一下自己
```

### 6. API服务

Ollama自带API服务：

```bash
# 启动API服务（默认端口11434）
ollama serve

# 调用API
curl -X POST http://localhost:11434/api/generate \
  -d '{
    "model": "qwen2.5:7b",
    "prompt": "什么是知识胶囊？",
    "stream": false
  }'
```
